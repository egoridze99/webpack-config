# Готовый конфиг для webpack  
Собранный webpack.config.js для дальнейшего использования  
___
## Для начала работы  
`npm install`  
___
## Подключенные модули  
1. babel
2. css.extract
3. css loader
4. sass loader
5. file loader  
6. copy  
7. normalize  
8. Минификаторы js и css
9. devserver
10. node static
___
## NPM команды  
### Для запуска dev-server  
`npm run devserv`  
### Для сбокри на production  
`npm start`
### Для запуска producrion сервера  
`npm run prodserv`
